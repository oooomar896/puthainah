# Coding Standards

- Use React 19 and Next.js conventions
- Use functional components and hooks
- Enforce ESLint (config in `eslint.config.mjs`)
- Follow Tailwind CSS utility-first patterns
- Avoid global side effects; prefer context/hooks
- Use TypeScript where applicable; keep types accurate
- Keep imports absolute with `@/` alias
- No inline secrets; use `.env` and `env.js`
